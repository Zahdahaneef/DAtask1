# Python Basics Mini Project

## 📌 Overview

This project is a beginner-friendly Python program that demonstrates fundamental programming concepts such as user input, variables, math operations, lists, and formatted output using f-strings.

It is designed as a step-by-step learning exercise to build a strong foundation in Python.

---

## 🚀 Features

* ✅ Hello World program
* ✅ User input and personalized welcome message
* ✅ Math calculations using user input
* ✅ List creation and access
* ✅ Simple bio-data display
* ✅ Clean formatting using f-strings

---

## 🛠️ Technologies Used

* Python 3

---

## 🧠 Learning Outcomes

* Understanding Python syntax
* Working with user input and variables
* Performing arithmetic operations
* Using lists and indexing
* Writing clean and readable code using f-strings
